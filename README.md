### 🔧 What I'm building

- **[seedance-prompt-forge](https://github.com/thoxakihiko/seedance-prompt-forge)** — a CLI + library that turns simple inputs (subject, mood, shot, lighting…) into clean, structured prompts for text-to-video models like **Seedance, Kling, Runway, and Veo**.
  ```bash
  npm install -g seedance-prompt-forge
  forge --subject "lone samurai" --mood tense --shot low-angle --style dark-anime
  ```
